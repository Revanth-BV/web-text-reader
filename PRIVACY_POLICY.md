# Privacy Policy for ReadIt Web Text Reader

ReadIt Web Text Reader is a browser extension that helps users read selected text aloud on any webpage.

## Data Collection
ReadIt Web Text Reader does not collect, store, transmit, or share any personal data or user information.

## Data Usage
All functionality runs entirely on the user’s device using the browser’s built-in Web Speech API. Selected text is processed locally and is never sent to any external servers.

## Third-Party Services
This extension does not use analytics tools, advertising networks, or third-party APIs.

## Permissions
The extension requires access to webpages only to detect user-selected text and provide reading controls. This access is used solely to deliver the core functionality.

## User Control
The extension activates only when the user explicitly selects text or uses the keyboard shortcut.

## Changes
If this privacy policy is updated, changes will be reflected on this page.

## Contact
For any questions or concerns, please open an issue on the GitHub repository.
